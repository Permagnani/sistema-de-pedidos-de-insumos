# Sistema de Pedidos de Insumos

Sistema web desenvolvido para auxiliar os responsáveis das lojas no envio de pedidos de insumos para a fábrica, tornando o processo mais prático, organizado e confiável.

## Sobre o projeto

O sistema foi criado para apoiar o processo de solicitação e separação de insumos entre lojas e fábrica. Para evitar esquecimentos no momento do pedido, a aplicação apresenta os itens do catálogo de forma visual e organizada, permitindo que o responsável selecione os insumos com mais segurança.

Além da montagem do pedido, o sistema também permite revisar as informações antes do envio e encaminhá-las diretamente via WhatsApp para o responsável pela separação, considerando os diferentes números utilizados de acordo com o dia.

Outro recurso da aplicação é a formatação do pedido recebido: ao colar a mensagem em um campo específico, o sistema organiza os dados em formato de tabela pronta para impressão, facilitando o controle de separação, o preenchimento do lote e o registro da quantidade separada.

## Funcionalidades

- Exibição dos itens disponíveis no catálogo
- Visualização das quantidades de cada insumo
- Montagem do pedido de forma prática
- Revisão antes do envio
- Envio direto via WhatsApp
- Suporte para responsáveis diferentes conforme o dia
- Conversão do pedido recebido em tabela para impressão
- Espaço para controle de lote e quantidade separada
